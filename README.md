# 🎬 Movie Recommendation System

A content-based Movie Recommendation System built using Python, NLP, and Machine Learning techniques.

---

## 🚀 Features
- Recommends similar movies instantly
- Uses cosine similarity algorithm
- NLP-based recommendation engine
- Dynamic user movie input

---

## 🛠 Technologies Used
- Python
- Pandas
- Scikit-learn
- NLP
- CountVectorizer

---

## 📊 Machine Learning Concepts
- Text Vectorization
- Cosine Similarity
- Content-Based Filtering

---

## ▶ How to Run

```bash
python src/recommender.py
```

---

## 📸 Sample Output

```plaintext
Enter movie name: Avatar

Recommended Movies:
Apollo 18
Tears of the Sun

...
```
